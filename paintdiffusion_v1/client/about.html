üéØ How It Works

1. Sketch Input

Draw your sketch directly on the HTML5 canvas or upload an existing image. The canvas provides intuitive drawing tools including undo/redo, color selection, and adjustable brush sizes.

2. Processing Mode Selection

Choose from multiple processing modes:

  • Scribble Processing: Interprets rough sketches with flexible guidance
  • Canny Edge Detection: Extracts and enhances edges for precise control
  • Line Art Enhancement: Optimizes clean line drawings

3. AI Generation

The system processes your sketch through ControlNet to create conditioning signals, which guide Stable Diffusion in generating images that match both your sketch structure and text prompt.

4. Enhancement (Optional)

Enable GAN refinement to upscale the generated image 4x using Real-ESRGAN, adding fine details and improving overall quality.

üìä Model Performance Evaluation

Our models have been rigorously evaluated using industry-standard metrics to ensure high-quality image generation. Below are the performance metrics for each component:

Understanding the Metrics

  • FID (Fr√©chet Inception Distance): Lower is better. Measures similarity to real images (0 = perfect match)
  • Inception Score: Higher is better. Evaluates image quality and diversity (max ~10)
  • Precision: Measure of generated image quality (0-1, higher is better)
  • Recall: Measure of diversity in generated images (0-1, higher is better)
  • LPIPS: Perceptual similarity metric (lower is better, 0 = identical)
